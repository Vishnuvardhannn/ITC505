Addendum

The primary goal of this project was to create an interactive and visually appealing "Lights Out" game that adhered to the given requirements while introducing additional features like a target moves counter, actual move counter, timer, and a restart button. Here's how the process unfolded:

Design and Structure:
The HTML layout was kept minimal and organized. Additional elements, such as the statistics section (target moves, moves, and timer), were added for user interactivity. A restart button was introduced for convenience, allowing users to restart the game at any point without refreshing the page.

Styling Choices:
A gradient background and vibrant colors (blue for active cells and pink for inactive cells) were chosen to create a visually modern and engaging design. Subtle hover effects and smooth transitions were implemented to enhance the user experience and make interactions intuitive.

Game Logic:
The game board is dynamically created with a 5x5 grid. A solveable random starting configuration was ensured by simulating random clicks rather than assigning states randomly. This guarantees every board is playable. The toggling logic, including neighbors, was carefully implemented and tested to ensure no edge cases caused errors.

Additional Features:

  • Target Moves: Displayed to challenge users to complete the puzzle efficiently.
  • Move Counter: Updates dynamically each time the user clicks on a cell.
  • Timer: Starts when the game loads, updating every second, and stops when the user wins.
  • Restart Button: Fully resets the board, moves, and timer for a fresh start.

Challenges and Solutions:
One challenge was ensuring the board was always solveable without manually testing configurations. This was addressed by simulating random clicks, which inherently maintain solveability. Additionally, implementing a proper restart function required careful resetting of all variables and states to avoid leftover data from the previous game.

Insights Gained:
This project reinforced the importance of modular design and the use of DOM manipulation for dynamic content creation. It also highlighted the significance of user-centric design in making a game enjoyable and accessible. This project not only met all stated requirements but also added several quality-of-life improvements, resulting in a polished and engaging game experience.
